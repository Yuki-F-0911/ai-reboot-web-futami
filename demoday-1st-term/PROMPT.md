# DEMO DAY 特設LP 制作指示書

## プロジェクト概要
AIリブートアカデミー第1期 DEMO DAY（成果発表イベント）のオンライン視聴参加を促進するための特設LPを作成してください。

## 作成するファイル
- index.html（1ファイル完結。HTML + CSS + 最小限のJS）
- 外部ライブラリ不要（vanilla HTML/CSS/JS）
- レスポンシブ対応（スマホファースト）

## コンテンツデータ
- content.json の内容をすべてLP上に掲載してください
- 各セクションの順序は content.json の section 番号順に従ってください

## デザイン・実装仕様
- design-spec.md を参照してください

## 重要な設計ポイント

### 1. タイムライン形式の受講生紹介（最重要セクション）
- 各受講生ブロックを DAY 1 → DAY 50 → DAY 100 の3ステップのタイムラインで構成
- DAY 1 と DAY 50 はリアルなコメント付きで表示
- DAY 100 は「???」としてブラックボックス化（鍵アイコン + COMING SOON 風の演出）
- ここが LP の核。期待感を煽り「当日見たい」と思わせるデザインにする

### 2. CTA の複数配置
- ファーストビュー、開催概要セクション後、フッター の3箇所に CTA ボタンを配置
- ボタンテキスト：「オンライン視聴を申し込む（無料）」
- リンク先：#（仮。後で Google フォーム URL に差し替え）

### 3. 全体のトーン
- 挑戦感・実践感・熱量がある
- ただの学校行事っぽくない、外部の人が見ても価値がわかる
- 抽象的すぎず、「何が見られるか」の輪郭は伝わる

### 4. 匿名対応
- 受講生は本名ではなくニックネームで表示
- 顔写真は使用しない。代わりに仮アバターアイコン（絵文字またはSVGイラスト）を使用
- 各受講生のフィールド（建築×AI 等）をラベルとして表示

### 5. スマホ最適化
- モバイルファーストで設計
- タイムラインはモバイルでも見やすい縦方向レイアウト
- CTA ボタンはスマホで親指が届く位置に

## 出力
index.html 1ファイルのみ。CSS はインラインまたは <style> タグ内に記述。
