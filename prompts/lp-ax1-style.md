# LP デザインシステム — Trust Style

このドキュメントは、特定のLPのデザイン言語（トンマナ・雰囲気・構造）を定義したものです。
内容やテーマに関わらず、このスタイルを適用することで一貫した品格を実現できます。

---

## 本質：3つのデザイン原則

### 1. 引き算の美学
- 装飾を足すのではなく、削ることで本質を際立たせる
- 色は2色（白×紺）＋アクセント1色のみ
- 余白は「何もない」のではなく「呼吸」

### 2. 静謐な緊張感
- 派手さより「重み」
- セールス感を消し、プレゼン資料のような品格
- 読者に「これは本気だ」と思わせる

### 3. 情報で魅せる
- ビジュアルより「構造」と「言葉」で説得
- 数字・事実・問いかけを武器にする
- イラストや装飾に頼らない

## カラーシステム

```css
--c-bg: #ffffff;           /* 純白 */
--c-bg-subtle: #f8fafc;    /* 微グレー（セクション区切り用） */
--c-ink: #1e3a5f;          /* ダークネイビー（主テキスト） */
--c-accent: #2563eb;       /* ロイヤルブルー（強調のみ） */
--c-sub: #64748b;          /* スレートグレー（補助テキスト） */
--c-border: #e2e8f0;       /* ライトグレー（境界線） */
```

**運用**:
- アクセント色は「数字」「タグ」「CTA」「ハイライト」のみ
- 背景は白/微グレーの2段階で区切る
- グラデーションはタイトルのみ許可

---

## タイポグラフィ

```css
--font-serif: "Shippori Mincho", serif;  /* 見出し */
--font-sans: "Zen Kaku Gothic New", sans-serif;  /* 本文 */
```

| 役割 | フォント | 特徴 |
|------|---------|------|
| 大見出し | serif | 大きく、詰めて、重く |
| 本文 | sans | 行間広め（1.8〜2）、読みやすく |
| 数字・強調 | serif | アクセント色で目立たせる |

---

## 余白の哲学

```css
--space-section: clamp(80px, 10vh, 150px);
--space-section-lg: clamp(120px, 20vh, 250px);
```

- **詰めない**。余白は贅沢品
- セクション間は大きく空ける
- 要素間の余白で「読みの呼吸」を作る

## 構造パターン

### ヒーロー
- 全画面（100vh）
- 中央に象徴的な1ワード or 短フレーズ
- 背景は微細なグリッド＋放射状グラデーション
- 日付や属性ラベルは角に斜め配置

### セクション区切り
- 背景色の切り替え（白↔微グレー）で区切る
- 線や装飾は使わない

### 数字の見せ方
- 大きく、serifで、accent色
- 横並びグリッド（4列→モバイル2列）
- 数字の下に短い説明

### リスト
- 左ボーダー（3px、accent色）
- 背景は半透明白
- hover時に右へ微スライド

### カード
- 番号は薄く大きく（opacity: 0.3）
- タグは小さなバッジで
- hover時にborder色変化＋影＋微浮上

### タイムライン
- 左に縦線
- 各項目の左にドット
- 内容はカード形式

### CTA
- 注意事項は別ボックスで先に
- 価格・条件は明確に
- ボタンはaccent背景、hover時に浮上

## テクスチャ

### ノイズオーバーレイ（全体）
```css
background-image: url("data:image/svg+xml,..."); /* fractalNoise */
opacity: 0.02;
mix-blend-mode: multiply;
```

### グリッドパターン（ヒーローのみ）
```css
background-image:
    linear-gradient(rgba(30, 58, 95, 0.03) 1px, transparent 1px),
    linear-gradient(90deg, rgba(30, 58, 95, 0.03) 1px, transparent 1px);
background-size: 60px 60px;
```

## アニメーション

### スクロール連動（fadeUp）
```css
.fadeUp {
    opacity: 0;
    transform: translateY(30px);
    transition: all 0.8s cubic-bezier(0.2, 1, 0.3, 1);
}
.visible { opacity: 1; transform: translateY(0); }
```
- 15%見えたら発火、一度きり

### ホバー
- カード: -2px浮上 + 影強化
- ボタン: 背景明化 + -2px浮上
- リスト項目: +5px右スライド

---

## コピーのトーン

- **断定**:「〜かもしれない」ではなく「〜だ。」
- **問いかけ**: 当事者意識を持たせる
- **数字**: 抽象論より具体的データ
- **短く**: ヒーローは1行、見出しは2行以内

---

## 避けるべきこと

- 3色以上の使用
- 丸みの強いUI（角丸は8〜16px）
- イラスト・絵文字・アイコンの多用
- 長文（箇条書き・カードで分割）
- セール文句（「今だけ！」「限定！」の乱用）
- 影や装飾の重ねがけ
