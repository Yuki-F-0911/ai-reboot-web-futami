# ニュース・お知らせセクション - コンテンツ構成

## 概要

ニュース・お知らせセクションは、AIリブートの最新情報を発信する重要なコンテンツです。トップページでの簡易表示と、詳細ページでの一覧表示の2つの形式で展開します。

## カテゴリー設計

### メインカテゴリー
1. **お知らせ** - サービスに関する重要な情報
2. **イベント** - 説明会、セミナー、ワークショップ情報
3. **メディア掲載** - 新聞、雑誌、Web媒体での紹介
4. **受講生の声** - 卒業生の成果や体験談
5. **コラム** - AI活用に関する知識・ノウハウ記事

### タグ設計
- #AIリブートアカデミー
- #AIリブート研修
- #リスキリング補助金
- #受講生募集
- #導入事例
- #AI活用Tips

## トップページでの表示

### 表示形式
- 最新3件を表示
- 日付、カテゴリー、タイトルのシンプルな構成
- 「すべて見る」リンクでニュース一覧ページへ

### サンプル表示
```
2025.01.15 【お知らせ】AIリブートアカデミー第3期生募集開始
2025.01.10 【メディア掲載】日経新聞にて当社の取り組みが紹介されました
2025.01.05 【イベント】無料オンラインセミナー「AI時代のキャリア戦略」開催
```

## ニュース一覧ページ

### ページ構成
1. **ヘッダー部分**
   - ページタイトル「ニュース・お知らせ」
   - カテゴリーフィルター
   - 検索ボックス

2. **一覧表示部分**
   - 1ページ10件表示
   - サムネイル画像（あれば）
   - 日付、カテゴリー、タイトル、概要文（100文字程度）
   - ページネーション

3. **サイドバー**
   - カテゴリー別アーカイブ
   - 月別アーカイブ
   - 人気記事ランキング

## 個別記事ページ

### ページ構成
1. **記事ヘッダー**
   - カテゴリー表示
   - 公開日
   - タイトル
   - アイキャッチ画像

2. **記事本文**
   - 見出し構造を持った本文
   - 画像、動画の埋め込み対応
   - 関連リンク

3. **記事フッター**
   - SNSシェアボタン
   - 関連記事表示（同カテゴリーから3件）

## 想定記事例

### お知らせ記事例
**タイトル**: AIリブートアカデミー第4期生募集開始！早期申込特典あり

**概要**: 2025年3月開講の第4期生の募集を開始しました。2月15日までにお申し込みの方には、特別教材をプレゼント。リスキリング補助金の活用で最大70%OFFで受講可能です。

### イベント記事例
**タイトル**: 【2/10開催】現役受講生が語る！AIリブート体験談セミナー

**概要**: 実際にAIリブートアカデミーで学んでいる受講生から、リアルな体験談を聞けるオンラインセミナーを開催。プログラムの詳細や、学習のコツなど、気になる疑問にお答えします。

### メディア掲載記事例
**タイトル**: 日本経済新聞にてAIリブートの取り組みが紹介されました

**概要**: 1月10日付の日本経済新聞朝刊にて、当社のAI人材育成プログラムが「実践型教育の新モデル」として紹介されました。記事では、メンター伴走型の特徴が評価されています。

### 受講生の声記事例
**タイトル**: 【卒業生インタビュー】営業職からAIコンサルタントへ転身した山田さん

**概要**: AIリブートアカデミー第1期卒業生の山田太郎さんにインタビュー。100日間のプログラムを経て、どのような変化があったのか、現在の仕事内容など、詳しくお聞きしました。

### コラム記事例
**タイトル**: ChatGPTを使った業務効率化の5つのポイント

**概要**: 日々の業務でChatGPTを効果的に活用するためのポイントを、実例を交えて解説。プロンプトの書き方から、セキュリティ面での注意点まで、実践的なTipsをご紹介します。

## 運用ガイドライン

### 更新頻度
- お知らせ：必要に応じて随時
- イベント：月2-3回程度
- メディア掲載：掲載時に随時
- 受講生の声：月1-2回
- コラム：週1回程度

### 記事作成のポイント
1. **タイトル**: 30文字以内で内容が分かるように
2. **概要文**: 100文字程度で要点をまとめる
3. **本文**: 見出しを使って読みやすく構成
4. **画像**: アイキャッチ画像は必須（1200×630px推奨）
5. **SEO**: 適切なメタデータの設定

### 記事の承認フロー
1. 担当者が原稿作成
2. 責任者がレビュー・承認
3. 公開設定（即時公開/予約公開）

## 技術要件

### CMS要件
- WordPress等の一般的なCMSを想定
- カテゴリー・タグ管理機能
- 予約投稿機能
- 画像アップロード・管理機能
- SEO対策機能（メタデータ設定）

### 表示要件
- レスポンシブデザイン対応
- 高速表示（遅延読み込み等）
- RSS配信対応
- サイト内検索対応

## KPI設定

### 測定指標
- 月間PV数
- 平均滞在時間
- 記事からのコンバージョン数
- SNSシェア数
- メルマガ登録数

### 目標値（6ヶ月後）
- 月間10,000PV
- 平均滞在時間2分以上
- 月間CV数20件以上
