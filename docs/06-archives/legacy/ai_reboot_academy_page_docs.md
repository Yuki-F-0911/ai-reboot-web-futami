> 「これまでの経験則が、もう通用しないかもしれない」  
> 「この変化のスピードに、正直ついていけるか不安だ」

そんな漠然とした不安は、決してあなただけが感じているものではありません。私たちは今、誰もが先の見えない「スーパーVUCA時代」の“ど真ん中”に立っているのです。

## AIリブートのコアコンセプト

「AIリブート」は、AI、特にChatGPTやGeminiなどの生成AIの活用を通じて、**企業と個人をアップデートし、イノベーションを起こす**というコンセプトで運営される事業です。  
これは、単なる教育事業ではなく、**AIを通じて人々と企業の可能性を解放する仕組み作り**を目指しています。

## 最も重要なメッセージと哲学

### 1. Will（意思）とウェルビーイング（Well-being）の実現

ウィルフォワードの根底にある理念は、**「世界平和」や「人類が人類と共存し、人類が地球と共存するサステナブルな未来」**の実現です。この実現のためには、**一人ひとりが自身の「Will（意思）」に向き合い、自分自身を幸せにすること**が不可欠であるとされています。

ウェルビーイングとは、身体的、精神的、社会的に満たされた状態を指し、目先の利益や二者択一に囚われず、複数の価値を統合・両立する「物質 AND」の視点を持つことが、真のウェルビーイング実現の鍵であるとされています。

AI時代において、AIが人間の多くのことを代行する中で、**人間が「本来何を果たすのか」という問い**に立ち返り、自身のWillに深く向き合い、自己認知を深めることが強調されています。  
働くことの概念が変わり、「働かなくなる」のではなく、「働く概念が変わる」という視点も提示されています。

### 2. 実践と実績による価値提供

単にAIの知識を「教える」だけでなく、受講者が実際にAIを「使えるようにさせる」こと、そして**実践を通じて結果を出し続けること**が重視されています。

講師陣は、現役のコンサルタントや生成AIスペシャリストが務め、**自身がAIを業務で使いこなし、サービス開発を行っている**という「実績」が、受講生への説得力につながると考えられています。

AIは常に進化するため、**常に最先端の技術を調査・学習し、それを自社で実践し成果を出す**ことが、コンサルティングや教育において重要であると述べられています。

### 3. 受講生を「顧客」ではなく「仲間」として捉えるエコシステム

AIリブートアカデミーは、受講生を単なる「顧客」としてではなく、**共に事業を創造する「仲間」として迎え入れる**という発想の転換を重視しています。

受講生が自己投資した費用（例: 15万円の自己負担分）を、将来的に講師やメンターとして活動することで「回収できる」ような仕組みを提示し、**長期的な協力関係**を築くことを目指しています。

## AIリブートの具体的な事業展開

AIリブートは、**個人向けのAIスクール事業「AIリブートアカデミー」**と、**企業向けのAI活用コンサルティング事業**の二つの柱で展開されます。

### 個人向け

- 経済産業省のリスキリング補助金対象講座として採択されており、国からの支援を受けられます。  
- **100日間で生成AIを使いこなせるようになるプログラム**で、単なるツール利用だけでなく、自身のWill実現のためのキャリア設計も伴走支援します。  
- 初期の2日間は集合研修（対面）で、個人とチームのAI活用環境構築に焦点を当てます。

### 法人向け

- 企業がAIを活用せずに成長を続けることが困難な時代において、変革を必要とする企業を支援します。  
- **採用、マーケティング、組織変革**といった分野でAIを活用し、イノベーションを支援します。  
- 特に、地方の中堅・中小企業をターゲットとし、「伝統文化×AI」「介護×AI」「農業×AI」「食×AI」など、各企業の事業内容に合わせたAI活用を提案し、企業価値を高めることを目指します。  
- 座学、実践、伴走型開発支援の三段構成で、社内にAIリテラシーと活用力を育み、チームでのAI作業環境構築を支援します。

最終的に、AIリブートは**「AIの時代に人間の価値を最大化する」**ことを使命とし、日本のAI教育のスタンダードを共に作り上げることを目指しています。

## 【100日間のロードマップ】自分だけの「Will」を見つけ、体現する旅

1. **AIリブートキャンプ (Kick-off合宿)**  
   - 全ての始まり。一生の仲間と出会い、徹底的に自分と向き合う。  
2. **1ヶ月目：Willの探求と課題設定**  
   - メンターとの対話を通じ、心の底から「やりたいこと（Will）」を探求。あなただけの挑戦課題を設定します。  
3. **2ヶ月目：AIと共に課題へダイブ**  
   - 生成AIを駆使し、試行錯誤を繰り返しながら課題解決に没頭。中間メンタリングで軌道修正をサポート。  
4. **3ヶ月目：アウトプットの磨き込み**  
   - 100日間の挑戦の軌跡を、人に伝わる形にまとめ上げていきます。  
5. **DEMO DAY：挑戦の発表**  
   - あなたの「Will」と「挑戦の物語」を、仲間と企業担当者の前で発表します。
