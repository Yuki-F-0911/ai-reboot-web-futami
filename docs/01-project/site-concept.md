# AIリブート Webサイト - 情報設計とコンセプト

## サイトコンセプト

### ミッション
「AIと人間が共存する未来において、個人と組織のWill（意思・目的）を明確にし、AIを活用してウェルビーイングな社会を実現する」

### コアバリュー
1. **Will-Driven（意思駆動）**: 個人の価値観と目的から始まる変革
2. **実践重視**: 座学だけでなく、実際の成果創出まで伴走
3. **人間中心**: AIはツールであり、人間の可能性を拡張する手段
4. **ウェルビーイング**: 幸福の総量を増やし、一人ひとりが豊かに生きる社会

## ターゲットペルソナ

### 個人向け（AIリブートアカデミー）
- **主要ペルソナ**: 30-50代のビジネスパーソン
- **課題感**: AI時代に取り残される不安、キャリアの方向性への迷い
- **ニーズ**: 実践的なAIスキル習得、自己のWill発見、新しいキャリアパス構築
- **期待する成果**: AI活用スキル習得、自己変革、キャリアチェンジ

### 法人向け（AIリブート研修）
- **主要ペルソナ**: 中堅・中小企業の経営者、DX推進責任者
- **課題感**: AI活用の必要性は理解するが、どう始めるか分からない
- **ニーズ**: 社内AI人材育成、実務での即活用、継続的なイノベーション創出
- **期待する成果**: チーム全体のAI活用力向上、業務効率化、新事業創出

## ユーザージャーニー

### 個人向けユーザー
1. **認知**: 「AI時代のキャリア不安」検索 → トップページ着地
2. **興味**: キャッチコピーとメッセージで共感
3. **検討**: プログラム内容確認、補助金情報チェック
4. **行動**: 資料請求または説明会申込み

### 法人向けユーザー
1. **認知**: 「企業AI研修」「DX人材育成」検索 → トップページ着地
2. **興味**: 実践型アプローチに魅力を感じる
3. **検討**: 導入事例確認、費用対効果を検証
4. **行動**: 問い合わせフォームから相談
