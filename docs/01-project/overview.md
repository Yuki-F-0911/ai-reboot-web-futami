### AIリブート事業の説明 

ウィルフォワード社が提供する「AIスクール事業」と「AIコンサルティング事業」は、総称して「**AIリブート**」と呼ばれます 。この事業は、ChatGPTやGeminiといった生成AIの活用を通じて、企業および個人のアップデートとイノベーションを促進することをコンセプトとしています 。

**AI進歩がもたらす変化と問いかけ** 
AIの進歩は著しく、これまでの常識を根本から変えつつあります 。この未来において、私たちはAIを単なるツールとして使いこなすだけでなく、AIが当たり前になった社会で**どのように生き、どのように働き、どのような価値を生み出すのか**が問われています 。

**ウィルフォワードの理念とAI** 
*   ウィルフォワードは創業以来、個人の「Will」（意思や未来）が社会を創造すると信じ、一人ひとりが自身のWillに向き合い、それを社会に提供することを使命としてきました 。
*   AIの時代において、AIが多くのことを人間に代わって行う社会では、**人間本来の役割**という原点に立ち返る問いが突きつけられています 。
*   AIに必死についていくだけでなく、AIと人類が共存する未来を創造するために、企業として、また個人としてAIとどのように向き合い、活用するのかを追求します 。
*   自身のWillに向き合い自己認知を深め、AIを最大限に活用することで、**ウェルビーイングな未来**を目指したいと考えています 。

**ウェルビーイングな未来の実現** 
社会が大きく変わる中で、以下の要素を通じてウェルビーイングな未来を目指します 。
*   **自己認知の深化とWillの明確化**: 自身のWillを知り、その実現方法を模索します 。
*   **最適な場所の追求**: これまでの画一的な成功モデル（良い学校、良い会社、出世、結婚、家庭構築など）ではなく、一人ひとりの多様な価値観とWillに基づき、自身のウェルビーイングを追求します 。
*   **AI活用による実現**: これまで不可能だったことや、一人では難しかったこと（例：アイデアの具現化、サポート不足による停滞）を、AIツール（ICT）の力を借りて実現していきます 。
*   **幸福の総量増加**: 他者から奪い合うのではなく、新エネルギー創出、効率的な食料生産技術、バイオ・医療の進化など、AIによるイノベーションを通じて**社会全体の幸福の総量を増やす**ことを目指します 。

**AIリブートの2つの事業** 
1.  **個人向けAIスクール事業「AIリブートアカデミー」** 
    *   **概要**: 経済産業省のリスキリング補助金事業として採択されており、国からの支援を受けられるスクールです 。
    *   **プログラム内容**: 100日間で生成AIを使いこなせるよう自己をアップデートします 。単にAIの使い方を学ぶだけでなく、**「どんな生き方をしたいのか」「どんなWillを持っているのか」「それを実現するためにどんなキャリアを歩むべきか」「どんな技術を磨くべきか」「どんな場所を目指すのか」「今何をすべきか」**といった自己分析とキャリア設計をキャリアコンサルタントが伴走しながら進めます 。
    *   **研修形式**: プログラムの冒頭2日間は、全国から受講生が集まる集合研修形式です 。これは、オンラインだけでは得られない「人の熱」や「チーム形成による創造性」を重視しているためです 。
    *   **講師陣**: ウィルフォワードの代表である成瀬拓也、青木亮、坂本琢磨といった、実際に生成AIを業務で使いこなし、アプリケーション開発などを行っている現役のコンサルタントや、特定の分野のAIスペシャリストが担当します 。キャリアコンサルタントもサポートします 。
    *   **実践型プログラム**: 2日間の研修後、100日間のオンラインプログラムを通じて、実践型のテーマに取り組みます 。ツールの使い方だけでなく、**「何のために、誰のために、誰に向けて、何をするのか」という全体像から具体的な指示（プロンプト）を作成する技術**を習得し、最終的にはデモデーで成果を発表します [新規録音 14.m4m, 新規録音 14.m4a]。

2.  **企業向けAI活用コンサルティング事業** 
    *   **対象**: 企業の規模に制限はありませんが、AIに取り組まなければ継続的な事業発展が難しいと認識している企業を対象とします 。特に、地方の中堅・中小企業など、素晴らしい価値を持ちながらも顧客到達や採用、時代適応に課題を抱える企業を重視しています 。
    *   **目的**: 企業がAIを使いこなせるようになるだけでなく、**社内でAIを実践できるスキルを育む**ことを目的とします 。
    *   **支援内容**: 伝統文化×AI、介護×AI、医療×AI、農業×AI、食×AIといった形で、企業の魅力的なイノベーションを創出する支援を行います 。具体的には、社内のAIリテラシーや活用力を高めるための研修プログラム（10時間を基準）を提供します 。
    *   **実践的な環境構築**: 生成AIツールだけでなく、統合開発環境（IDE）を活用し、個人が最適な開発を行える環境を整備します 。ウェブサイト制作、マーケティングコンテンツ自動生成、画像生成、スライド資料作成、動画生成、チャットボット生成、音声生成など、事業に関連性の高いテーマを実際に作成を通じて習得します 。
    *   **応用力の育成**: 特定のツールを使う力ではなく、あらゆるツールに応用が効く**応用力**を鍛えることに重点を置いています 。
    *   **チームでのAI活用**: 後半では、個人だけでなくチームでのAI開発や作業環境構築に取り組みます 。企業の共通資産（顧客データ、社内情報など）を一元管理し、共通でアクセス可能なリポジトリを構築することで、チーム全体のAI活用を効率化します 。
    *   **OJTメンター**: 実践力を磨くために、OJTメンターがプロジェクトをサポートし、業務改善やイノベーション創出を支援します 。

**最終的な目標** 
これら個人向け・法人向けのAIリブート事業を通じて、個人および組織のアップデートとイノベーションを支援し、**社会がより豊かになり、一人ひとりがウェルビーイングな社会を実現すること**を目指します 。
