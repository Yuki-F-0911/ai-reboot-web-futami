# 新トップページ コンテンツ＆メッセージング設計案

## 1. コンセプト

### ターゲット
- **共通像**: 「現状維持ではいけない」という変革への意志を持つ個人と組織。
  - **個人**: キャリアや生き方に漠然とした不安や迷いを抱える30〜50代。
  - **法人**: 事業の未来に危機感を持ち、変革の必要性を感じている経営者・リーダー。

### コア体験
- AIの進化という大きな変化を、「脅威」から「自己実現の最大の好機」へと視点転換させる。
- 自分の内なる「Will」こそが、AI時代を生き抜く最強の羅針盤であると気づかせる。

### トーン＆マナー
- 先進的だが、温かい。
- 不安に寄り添い、希望を提示する。
- 専門的だが、平易な言葉で語る。

---

## 2. コンテンツ構成案

### セクション1: ヒーロー (Above the Fold)

- **目的**: 訪問者の心を掴み、自分ごと化させる。不安を希望の入り口に変える。
- **メインキャッチコピー**:
  - **AI時代の羅針盤は、あなたの「Will」にある。**
- **サブキャッチコピー**:
  - AIを学ぶのではない。AIで、"なりたい自分"を創るのだ。
- **ビジュアル**:
  - 中央に、内側から光を放つ人のシルエット。その周りを温かみのある光の粒子（AI）が巡り、未来への道筋を照らすインタラクティブなアニメーション。
- **CTA (Call To Action)**:
  - **プライマリボタン**: `[ AIで、未来の扉を開く ]` (共通の入り口)
  - **セカンダリリンク**:
    - `個人の方はこちら >`
    - `法人の方はこちら >`

### セクション2: プロブレム (共感)

- **目的**: 個人と法人が抱える共通の根源的な問いを投げかけ、深く共感を得る。
- **ヘッドライン**: `AIが、あなたに問いかけている。`
- **コンテンツ (左右対比)**:
  - **個人の問い (左側)**:
    - `このままで、私の価値は保たれるのか？`
    - `本当にやりたいことは、今の仕事の延長線上にあるのか？`
    - `変化の波に、ただ飲まれるだけでいいのか？`
  - **組織の問い (右側)**:
    - `このままで、我々の事業は存続できるのか？`
    - `本当に提供すべき価値は、今のサービスの延長線上にあるのか？`
    - `変化の波を、ただ眺めているだけでいいのか？`
- **クロージングメッセージ**: `その答えは、外にはない。あなたの、我々の「内」にある。`

### セクション3: ソリューション (AIリブートの哲学)

- **目的**: AIリブートの独自性「Will-centric」を提示し、他社との違いを明確にする。
- **ヘッドライン**: `スキルより、Willを。戦術より、目的を。`
- **コンテンツ**:
  - **コンセプト図**:
    1.  中心: **Will (個人の意思 / 組織の理念)**
    2.  周り: **AI (強力な実現ツール)**
    3.  外側: **TRANSFORMATION (自己変革 / 事業変革)**
  - **テキスト**:
    - 私たちは、AIの使い方を教えるだけのスクールではありません。
    - 私たちは、AIツールを導入するだけのコンサルではありません。
    - あなたの、そして貴社の「Will」を解放し、AIという翼を授けるパートナーです。
    - **AIリブートとは、AIで「本当の自分・本当の価値」を再起動すること。**

### セクション4: サービス紹介 (具体的な解決策)

- **目的**: 2つのサービスが同じ哲学に基づいていることを示し、ユーザーを適切なページへ誘導する。
- **ヘッドライン**: `あなたの「再起動」を、ここから。`
- **コンテンツ (左右2分割カード)**:
  - **左カード: AI REBOOT ACADEMY (for Individuals)**
    - **タイトル**: `個人のための、100日間の自己変革プログラム`
    - **キャッチ**: `「なりたい自分」を発見し、AIで実現する力を手に入れる。`
    - **キーワード**: `#キャリアチェンジ #自己実現 #リスキリング補助金`
    - **CTA**: `[ アカデミーの詳細を見る ]`
  - **右カード: AI REBOOT FOR BIZ (for Corporations)**
    - **タイトル**: `組織のための、実践型AI導入・人材育成`
    - **キャッチ**: `「ありたい姿」を定義し、AIで事業を飛躍させる。`
    - **キーワード**: `#DX推進 #イノベーション #生産性向上`
    - **CTA**: `[ 法人研修の詳細を見る ]`

### セクション5: ストーリー/思想 (信頼と共感の深化)

- **目的**: なぜウィルフォワードがこの事業を行うのか、その背景にある思想を語り、信頼を醸成する。
- **ヘッドライン**: `私たちが「AIリブート」に込めた想い`
- **コンテンツ**:
  - ウィルフォワード創業以来の「個人のWillが社会を創造する」という信念。
  - AI時代は、その信念を加速させる最大のチャンスであること。
  - AIによって、これまで諦めていた個人の挑戦や、リソース不足に悩む企業の変革が可能になる。
  - 私たちは、テクノロジーの力で「誰もが自分のWillを実現できる社会」を創りたい。そのための具体的なアクションが「AIリブート」である。

### セクション6: CTA (行動喚起)

- **目的**: 最後のひと押し。具体的な次のステップへ導く。
- **ヘッドライン**: `さあ、あなたの手で、未来をリブートしよう。`
- **コンテンツ**:
  - 不安や疑問、何でもお聞かせください。
  - AIの専門家であり、キャリアと組織のプロが、あなたの最初の対話相手になります。
- **CTA**:
  - **プライマリボタン**: `[ まずは無料で相談してみる（個人/法人共通） ]`
  - **セカンダリリンク**: `[ 最新情報・活用事例を受け取る（メルマガ登録） ]`
