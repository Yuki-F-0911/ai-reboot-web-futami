# AIリブートアカデミー トップページデザインコンセプト

作成日: 2025-01-05
作成者: Claude

## 🎯 デザインの目的

「読者に最後まで読んでもらい、心に何かを残す」

### 核心的な課題
- 長文テキストを最後まで読ませる
- 説教くさくならずに、深いメッセージを伝える
- 読者の内なる「Will」を呼び覚ます

## 🌟 コンセプト：「静かな覚醒の物語」

### ファーストビュー：「違和感の可視化」

**コンセプト**
毎日感じている小さな違和感を、視覚的に表現する。説明するのではなく、感じさせる。

**ビジュアルアイデア**
1. **時計の針が逆回転するアニメーション**
   - 「毎日、同じことの繰り返し」を暗示
   - スクロールで針が正常に動き始める

2. **ぼやけた鏡のような背景**
   - 自分の姿がはっきり見えない状態
   - スクロールで徐々にクリアになる

3. **静かなノイズエフェクト**
   - 内なる声が聞こえそうで聞こえない状態
   - テキスト「毎日、同じことの繰り返し。」が浮かび上がる

## 📜 スクロール体験設計

### 1. 序章：共感から始まる（0-20%）
- **演出**: 薄暗い背景から徐々に明るくなる
- **テキスト**: 大きめのフォントで、ゆっくりフェードイン
- **狙い**: 「これ、私のことだ」と感じさせる

### 2. 第一章：問いかけの深化（20-35%）
- **演出**: テキストが中央に集まり、読者の視線を引き込む
- **インタラクション**: 「なぜ」の部分がハイライトされる
- **狙い**: 自分への問いかけを促す

### 3. 第二章：可能性の提示（35-50%）
- **演出**: 背景に抽象的なAIビジュアル（粒子、ネットワーク）
- **テキスト配置**: 左右交互に配置し、リズムを作る
- **狙い**: 未来への期待感を演出

### 4. 第三章：関係性の転換（50-65%）
- **演出**: 「共犯者」の文字が手書き風にアニメーション
- **背景**: 温かみのある色調へシフト
- **狙い**: 親近感と信頼感を醸成

### 5. 第四章：変化の実感（65-80%）
- **演出**: 具体的な変化を視覚化（アイコン、イラスト）
- **レイアウト**: カード形式で読みやすく
- **狙い**: 「自分もこうなれる」という実感

### 6. 最終章：決断の瞬間（80-100%）
- **演出**: すべてが静かになり、最後のメッセージに集中
- **CTA**: 控えめだが存在感のあるボタン
- **狙い**: 自然な行動への誘導

## 🎨 ビジュアルスタイル

### カラーパレット
```
背景の変化：
- 序章: #1a1a1a (深い黒) → #2a2a2a
- 中盤: #2a2a2a → #f5f5f0 (温かみのある白)
- 終盤: アクセントに金色 (#d4af37)

テキスト：
- メイン: #ffffff → #333333 (背景に応じて反転)
- 強調: #d4af37 (金色、「Will」など重要な概念)
- サブ: #666666
```

### タイポグラフィ
```
見出し: 游明朝体 or Noto Serif JP
- 日本の美意識を表現
- 静かな力強さ

本文: Noto Sans JP
- 高い可読性
- 長文でも疲れない

強調: 手書き風フォント（部分的に使用）
- 人間味、温かみの演出
```

### レイアウト原則
1. **余白を贅沢に使う**
   - テキストを詰め込まない
   - 思考する時間を与える

2. **視線の流れを設計**
   - Z字、F字の読みパターンを意識
   - 重要なメッセージは視線の交点に

3. **リズムと緩急**
   - 短文と長文の組み合わせ
   - 画像やアニメーションでの休憩

## 🎬 アニメーション設計

### 基本方針
- **控えめだが印象的**：派手さではなく、意味のある動き
- **読書を邪魔しない**：テキストの可読性を最優先
- **感情に訴える**：機械的ではなく、有機的な動き

### 具体的な演出
1. **パララックススクロール**
   - 背景要素がゆっくり動く
   - 奥行き感の演出

2. **テキストアニメーション**
   - 重要な言葉が浮かび上がる
   - 読むペースに合わせた出現

3. **トランジション**
   - 章の切り替わりを優雅に
   - フェード、ブラー、スケール

## 💡 インタラクション設計

### マイクロインタラクション
1. **ホバーエフェクト**
   - 「Will」などキーワードにカーソルを合わせると説明
   - さりげない光のエフェクト

2. **プログレスインジケーター**
   - 画面端に読了率を表示
   - 章の区切りがわかる

3. **読了時間表示**
   - 「この物語は約5分で読めます」
   - 心理的ハードルを下げる
