生成AI活用力研修「AIリブート」
（通称：AIリブート）

生成AIツールの使い方を学ぶだけではなく、生成AIツールを使いこなすためのマインドセットとスキルを身につけることで、研修後も持続的に生成AI活用で企業と個人をアップデートしつづけるためのプログラム。

企業が直面するAI時代の課題

AIの急速な進化は既存の常識を大きく変え、
企業がAIを活用せずに成長を続けることは困難な時代です。

  • AI導入の迷い: AIを活用せずに成長を続けることは困難な時代。でも何から始めればいいか分からない。
  • 成長の停滞: 従来のやり方では限界を感じている。新しい価値創造の方法を模索している。
  • 人材の課題: AI時代に対応できる人材の採用も育成も難しい。組織全体の変革が必要。

プログラムの内容と特徴

インプットとアウトプットの実践型研修プログラム

生成AI活用を学ぶだけではなく、生成AIを活用する環境を整えて、個人だけではなく、チームとして生成AI活用を業務に組み込みます。

AIリブート研修（12時間）

AIの活用方法や編集スキルの習得を目的とした対面での集中研修。

DAY1：個人作業環境の構築

  • 生成AI活用で実現できること
  • 生成AIが変える未来
  • 生成AI活用事例
    • 資料作成／画像生成／スライド作成／動画生成／音楽生成／
      データ分析／ダッシュボード開発／自動化／チャットボット開発 等
  • 個人作業環境（AI搭載IDEおよびLLMチャットツール等）の構築
  • 生成AI活用実践（実際に個別の業務にAIを活用する）

DAY2：チーム作業環境の構築

  • チームでAI活用にレバレッジをかける
  • チームで共有できる生成AIリポジトリの構築
  • ナレッジ共有や情報管理の仕組みを実装
  • チーム全体でAIを活用した共同作業を体験

実践フォロー研修（1時間 × 6回）

AIリブート研修で学んだスキルと整えた環境を使って、実際に業務に取り組みながら、さらなるAI活用スキルの向上を目指すサポートプログラム。
実践フォロー研修を通じて、受講者の進捗確認・課題解決・応用事例の共有を行い、生成AI活用を業務に根付かせることを目指します。

プログラムの実施期間とスケジュール

プログラム全体の推奨期間：3ヶ月以内

- AIリブート研修：プログラム連続2日間（企業の事情により複数回の分割は可能）

- 実践フォロー期間：AIリブート研修後から3ヶ月以内を目安（週1回頻度で実施を推奨）
